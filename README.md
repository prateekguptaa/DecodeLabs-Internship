# Nature Explorer

A simple and responsive static webpage built using **HTML5** and **CSS3**. This project showcases a clean webpage layout with semantic HTML elements, navigation, content sections, an image gallery, and a footer.

## Features

- Semantic HTML5 structure
- Responsive and clean design
- Navigation bar
- About section
- Nature image gallery
- Contact section
- Footer with copyright information

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
Nature-Explorer/
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

## Project Overview

The webpage includes:

- **Header** with website title and introduction
- **Navigation Menu** for easy section access
- **About Section** with information about nature
- **Gallery Section** displaying nature images
- **Contact Section** with sample contact details
- **Footer** containing copyright information
